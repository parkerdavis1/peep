# Peep

![Peep](src/assets/peep2_witheye.svg)

An web application to perform common edits for audio before uploading to eBird.

- Trim (with 3 second buffer) to target audio
- Normalize to -3dB
- Minimal high-pass filter
- Add 1 second fade in/out

Mobile users: Prevents users from having to download audio from their phone onto their computer to edit before uploading to eBird.

Desktop users: Simplifies and expidites common editing tasks.
